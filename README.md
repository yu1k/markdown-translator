# markdown-translator

Markdown で書いたドキュメントをHTMLに変換するスクリプトです。

## 環境

- OS
  - Ubuntu18.04

- シェル
  - bash

- 言語
  - JavaScript

```
$ node -v
```

v10.14.2

## How to use

```
$ yarn install
```

パッケージをインストール。

```
$ node index.js hoge.md fuga.html
```

で MarkdownファイルからHTML を生成。
